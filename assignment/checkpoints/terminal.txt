# 1 - What is the command for listing the files in the current folder (directory)?

    The command is 'ls'.

# 2 - What is the command for changing directories?

    The command is 'cd' then the name of the directory.

# 3 - What is the command for going "up" a folder (to a parent directory)?

    The command is 'cd ..'

# 4 - What is a quick way to create a new folder?

    Use the command 'mkdir' with the name of the new directory.

# 5 - What output would you expect when you run the command 'ls' in an empty folder? What about a folder containing files?

    Using 'ls' in an empty folder will show nothing at all. Using this command in a folder with files will show the content of all the files and directories in it.

# 6 - What is a quick way to create a new file?

    Use command 'touch' along with a file name and the intended file extension.

# 7 - Run 'git --version' in your terminal. What is the output?

    The terminal says 'git version 2.15.0'.

# 8 - Why would you run the command 'git status'?

    One would use 'git status' inside a folder that has a git repository to see the status of all the files being kept track of. It will display if any files have been modified, if any files are staged to commit, or if everything is up to date.

# 9 - What does the command 'git add .' do?

    The command 'git add .' stages any files that have been modified for a commit.

# 10 - What is the command to push your committed changes to your repo on GitHub?

    The command to use is 'git push -u origin master'
